49 ! TO COMPUTE THE VECTOR ROTATIONS, THE OPTIONAL ARGUMENTS
50 ! 'SROT' AND 'CROT' MUST BE PRESENT. TO COMPUTE THE MAP
51 ! JACOBIANS, THE OPTIONAL ARGUMENTS 'XLON', 'XLAT',

58 ! 97-10-20 IREDELL INCLUDE MAP OPTIONS
59 ! 1999-04-27 GILBERT CORRECTED MINOR ERROR CALCULATING VARIABLE AN

235  dlon=mod(rlon(n)-orient+180+3600,360.)-180
