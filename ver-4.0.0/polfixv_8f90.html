THIS SUBPROGRAM AVERAGES MULTIPLE POLE VECTOR VALUES ON A LATITUDE/LONGITUDE GRID.

BITMAPS MAY BE AVERAGED TOO. VECTORS ARE ROTATED WITH RESPECT TO THEIR LONGITUDE.

Parameters
NM- INTEGER NUMBER OF GRID POINTS
NX- INTEGER LEADING DIMENSION OF FIELDS
KM- INTEGER NUMBER OF FIELDS
RLAT- REAL (NM) LATITUDES IN DEGREES
RLON- REAL (NM) LONGITUDES IN DEGREES
IB- INTEGER (KM) BITMAP FLAGS
[out]LO- LOGICAL*1 (NX,KM) BITMAPS (IF SOME IB(K)=1)
[out]UO- REAL (NX,KM) U-WINDS
[out]VO- REAL (NX,KM) V-WINDS
